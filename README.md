产品概述

API接口文档

基础查询接口
`POST /api/v3/query`

请求参数
```json
{
  "question": "题目内容",
  "platform": "chaoxing",
  "options": ["A", "B", "C"],
  "type": "single_choice"
}
```

成功响应
```json
{
  "code": 200,
  "data": "这是答案"
}
```
